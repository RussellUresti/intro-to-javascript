There are 6 basic data types in JavaScript:

  • numbers: a numeric value
  • strings: text
  • booleans: true or false
  • functions: a block of wrapped code
  • arrays: a collection of data types
  • objects: a collection of data types... wait a minute...

A variable is a bucket or container for data.

Statements and Expressions

          

// Assign 20 to myVar
var myVar = 20;


// Assign 200 to myVar
var myVar = 20 * 10;

http://jsbin.com/zasiz/3/edit

A statment is a line of code that says "do this".

An expression is a piece of code that results in a value.

Statements can contain expressions, but expressions cannot contain statements.

Conditionals and Loops

          

// if conditional
if(1 > 0) {
  // do stuff
}

// switch conditional
switch(myVar) {
  case 0:
    // do stuff
    break;
  case 1:
    // do other stuff
    break;
}


// for loop
for(i = 0; i < 100; i++) {
  // do something 100 times
}

// do ... while loop
var i = 0;
do {
  // do stuff
  i++;
} while (i < 100);

http://jsbin.com/zasiz/4/edit

A conditional evaluates an expression and executes different code based on the result.

A loop repeatedly executes a block of code until a specific condition is met.

Ternary Statement

          

var myVar = 1 > 0 ? 'yes' : 'no';

// variable assignment
// = (conditional to test) 
// ? (value to assign if true) 
// : (value to assign if false)

http://jsbin.com/zasiz/5/edit

A ternary is an assignment statement that also evaluates a conditional to apply a value.

Array

http://jsbin.com/zasiz/9/edit

An array holds a collection of items in a numerical sequence.

Arrays start their numerical sequence at 0.

To access a specific value in an array, the syntax is myArray[index] where index is a positive integar or zero.

In JavaScript, an array is actually just a special type of object that uses numerical keys.

Functions

          

// define a function
function myFunc() {
  // execute code when invoked
}

// invoke the function
myFunc();


// define a function
function myFunc(param) {
  // execute code when invoked
}

// invoke the function
myFunc(10);


// define a function
function myFunc(param) {
  // execute code when invoked
  return param * 2;
}

// invoke the function
myFunc(10);

http://jsbin.com/zasiz/6/edit

A function is a block of code that executes only when invoked or called.

Functions can accept values as parameters.

Function can return a value when it finishes executing.
